# v1.0.0

First public release of Word PDF Watermark for macOS.

## Highlights

- Product-style bilingual README
- Demo assets for GitHub: hero image, workflow graphic, animated GIF
- Drag-and-drop `.app` build pipeline
- Vendored Python dependencies inside the app bundle (`pypdf`, `reportlab`)
- Reproducible release packaging script with SHA256 checksum output
