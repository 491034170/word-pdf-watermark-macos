Workflow

三步完成：选文件 → 输水印 → 导出结果

不管你是从命令行调用，还是把文件拖到 App 上，核心流程都保持一致。

01 · Select files 选择 Word / PDF / RTF 文件，支持多文件批量处理。
02 · Enter watermark 输入中文或英文水印文字，支持“仅供内部使用”“审阅版”等常见场景。
03 · Export PDF 在原文件旁生成 *_带水印.pdf，完成后即可直接发送或归档。
Word / RTF conversion requires LibreOffice · PDF watermarking works directly

FAQ

下载之前，别人通常会问的 4 个问题

把这些问题先解释清楚，落地页就更像正式产品页，而不是“源码仓库的附属说明”。

需要 LibreOffice 吗？

只有在输入是 Word / RTF 时才需要。纯 PDF 加水印不依赖 LibreOffice。

Release `.app` 能不能直接用？

可以。它已经自带核心脚本和 Python 依赖；目标机器仍需要具备 python3，这样打包后的启动器才能正常运行。

输出文件会放到哪里？

默认生成在原文件旁边，命名为 *_带水印.pdf，不强行改你的文件组织方式。

能不能继续二次开发？

可以。仓库里保留了源码、构建脚本、Release、MIT License 和 CI，后续继续做成更完整的 macOS 工具也顺手。
